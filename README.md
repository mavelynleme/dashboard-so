# Notas App API

API REST simples feita com Node.js e Express.

## Como executar

```bash
npm install
node index.js
```

A API usa a porta definida em `process.env.PORT`. Em ambiente local, caso essa variavel nao exista, usa a porta `3000`.

## Rotas

- `GET /` retorna uma mensagem confirmando que a API esta funcionando.

## Render

- Build Command: `npm install`
- Start Command: `node index.js`
